    • jdb 是一个能够在命令行中交互式的调试 java 应用程序的工具

    语法

    例子：

    1 jdb [options] main_class_name

    通过 jdb 启动一个java应用并在应用上开启一个 debug session

    2 jdb [options] -attach <address>

    运行 jdb 连接 一个独立运行的 java 应用并在应用上开启一个 debug session

    注意事项

    为了在调试过程中所有的调试命令都能够使用，调试的目标应用必须通过 “-g” 选项编译将生成的调试信息写入到 class 文件中，例如
    源文件内容、行号、变量信息等。

    调试 Running Applications

    In JDK 1.5, both “jdb” and “java” have been enhanced to use JPDA (Java Platform Debugger Architecture) technology to give you the following options

    Debugger Target
    Option jdb java
    1 Shared memory client Shared memory server
    2 Shared memory server Shared memory client
    3 Socket client Socket server
    4 Socket server Socket client

    shared memory 选项要求debugger 和 target application 在一个机器上，socket 选项允许远程运行

    java -agentlib:jdwp的使用帮助：

    suspend 如果为 y，目标 VM 将暂停，直到调试器应用程序进行连接。

    补充常用词：
    • JDI — Java 调试接口（Java Debug Interface）
    • JDT — Java 开发工具（Java Development Tools）
    • JDWP — Java 调试网络协议（Java Debug Wire Protocol）
    • JPDA — Java 平台调试器架构（Java Platform Debugger Architecture）
    • JVM — Java 虚拟机（Java Virtual Machine）
    • JVMDI — JVM 调试接口（JVM Debug Interface）
    • JVMTI — JVM 工具接口（JVM Tool Interface）
    • VM — 虚拟机（Virtual Machine）
